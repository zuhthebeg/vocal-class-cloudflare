- [ ] 컬러 팔레트 확정

  - [ ] Text Primary: `#4A4A4A` (다크 그레이)
  - [ ] Text Secondary: `#9E9E9E` (미디엄 그레이)
  - [ ] Border: `#E8E8E8` (라이트 그레이)

  - [ ] 8px grid system (8, 16, 24, 32, 40, 48, 64, 80px)

  - [ ] Icon Button: 원형, 아이콘만
  - [ ] 크기: sm, md, lg
  - [ ] 상태: hover, active, disabled, loading

- [ ] 모달 & 드로어
